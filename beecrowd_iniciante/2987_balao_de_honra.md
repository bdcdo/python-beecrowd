## Descricao

Dada uma letra do alfabeto, informe qual a sua posição.

## Entrada

Um único caracter **L**, uma letra maiúscula ('A'-'Z') do alfabeto.

## Saida

Um único inteiro, que representa a posição da letra no alfabeto.

## Exemplo

**Entrada:**
```
C
```

**Saida:**
```
3
```
